# Play Console - Data Safety Form Answers

## Data Collection
Does your app collect or share any of the required user data types? YES

### Location
- Collected: YES (approximate and precise)
- Shared with third parties: NO
- Required or optional: OPTIONAL (feature works without it)
- Processing purpose: App functionality (nearby stop detection, only on-device)

### All other data types: NOT COLLECTED
- No name, email, phone, address
- No financial info
- No health info
- No messages
- No photos/videos
- No audio
- No files
- No calendar/contacts
- No identifiers (no advertising ID used)
- No app activity beyond in-app navigation (not transmitted)
- No web browsing
- No app info/performance (no crash reporting SDK)

## Security Practices
- Data encrypted in transit: YES (HTTPS for GitHub sync)
- Data encrypted at rest: NO (SQLite on-device, standard Android storage)
- Users can request deletion: YES (cache clear in Settings removes local data)
- App follows Families Policy: NO
