# YBS Guide v1.0.0 — Release Notes

## What's New
- Complete Yangon Bus Service (YBS) route guide
- 139 validated YBS routes with bilingual names (English + မြန်မာ)
- Interactive Yangon map with bus stop markers
- GPS-based nearby stop detection
- Search by route number, stop name, or destination (English and Burmese)
- Offline support — works without internet
- Favorites for quick access to frequent routes
- Quick Access shortcuts for Home and Work routes
- Route schedules with departure times
- Auto-update route data without app reinstall

## Technical
- Flutter 3.x | Android min SDK 24 | Target SDK 34
- SQLite local database | OpenStreetMap tiles
- GitHub-based remote data sync
